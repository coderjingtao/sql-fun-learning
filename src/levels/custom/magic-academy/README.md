## Magic Academy

Imagine a magic academy where many students learn and take exams in different subjects. Please design a table named `magic_scores` to record the exam scores of each student in different subjects. The table fields are as follows:

- `student_id`: Student ID, uniquely identifies each student.
- `student_name`: Student name.
- `subject_id`: Subject ID, uniquely identifies each subject.
- `subject_name`: Subject name.
- `score`: The student's exam score in that subject.

Please write a SQL query to output each student's ID (`student_id`), name (`student_name`), subject ID (`subject_id`), subject name (`subject_name`), exam score (`score`), and the student's score ranking in each subject (`score_rank`), sorted by score from high to low.
